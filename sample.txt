## 🔹 9. Prime element check in a list

**Program:**

```python
def isPrime(n):
    if n <= 1:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True

lst = [2, 3, 4, 5]
for x in lst:
    print(x, isPrime(x))
```

**Conclusion:**
Checks primality of list elements.

## 🔹 12. Multilevel inheritance

**Definition:**
Inheritance where a class derives from another derived class.

**Program:**

```python
class A:
    def showA(self):
        print("A")

class B(A):
    def showB(self):
        print("B")

class C(B):
    def showC(self):
        print("C")

obj = C()
obj.showA()
obj.showB()
obj.showC()
```

**Conclusion:**
Demonstrates multilevel inheritance.
